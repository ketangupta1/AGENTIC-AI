### What is multimodal agent
- A multimodal agent is an AI agent that can understand, reason over, and act using multiple types of data (modalities) instead of only text.
- Like cursor is a developer agent which can read logs, Looks at screenshots (image), Checks metrics (structured data), Fixes code via Git tools.
- Gpt-3.5 turbo can take only text in input and gives text in output so its not a multimodal agent.
- Gpt-4o mini can take text as well as image in the input, so its a multi modal agent.
